# Tile Layer

Console graphics editor written in x86 assembly.

## Build

Requirements:
- DOSBox: https://www.dosbox.com
- DLINK.EXE from Adam Seychell's DOS 32 extender (dos32v33.zip): https://hornet.org/cgi-bin/scene-search.cgi?search=DOS%2032
- Turbo Assembler 4.0 (TASM32.EXE or TASM.EXE): https://www.retro-computing.com/Archives/downloadfile.php?fn=Archives/MSDOS/Files/BTASM40.ZIP&archivename=MSDOS&fnsize=23.9%20MB

Run MAKE.BAT. If all goes well, it will produce TLAYER.EXE.

## Run

Requirements:
- DOS32.EXE from Adam Seychell's DOS 32 extender (dos32v33.zip): https://hornet.org/cgi-bin/scene-search.cgi?search=DOS%2032

TLAYER.EXE path/to/rom
